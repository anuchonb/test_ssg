🔹 1. แนะนำระบบ

CRM Condo System เป็นระบบบริหารจัดการลูกค้าและสินเชื่อสำหรับธุรกิจขายคอนโด ออกแบบมาเพื่อรองรับการทำงานตั้งแต่รับลูกค้าใหม่ ไปจนถึงการโอนกรรมสิทธิ์

คุณสมบัติหลัก

  • ✅ เก็บข้อมูลลูกค้าและค้นหาอัตโนมัติด้วยเบอร์โทร
  • ✅ จัดการเคสตั้งแต่สร้างจนถึงอนุมัติ
  • ✅ ติดตามลูกค้าด้วย Timeline
  • ✅ ตรวจสอบคุณภาพ KPI
  • ✅ จัดการเอกสารและอัปโหลดไฟล์
  • ✅ ส่งธนาคารและบันทึกผลอนุมัติ
  • ✅ ปิดหนี้ จำนอง ตรวจห้อง
  • ✅ Dashboard สำหรับผู้บริหาร
💡 การไหลของงาน (Workflow):
ลูกค้า → เคส → ติดตาม → KPI → Pre-Approve → เอกสาร → ส่งธนาคาร → อนุมัติ → ปิดหนี้ → จำนอง → ตรวจห้อง → โอน

🔹 2. การเข้าสู่ระบบ

ขั้นตอนการเข้าสู่ระบบ

1 เปิดเว็บเบราว์เซอร์และเข้าสู่ URL ของระบบ

2 กรอก อีเมล และ รหัสผ่าน ที่ได้รับจากผู้ดูแลระบบ

3 คลิกปุ่ม "เข้าสู่ระบบ"

🔑 บัญชีตัวอย่าง:
Admin: admin@company.com
Admin Page: somsri@company.com
KPI: wichan@company.com
Support: mana@company.com
รหัสผ่านเริ่มต้น: 123456
⚠️ คำเตือน: ควรเปลี่ยนรหัสผ่านหลังเข้าสู่ระบบครั้งแรก

การออกจากระบบ

คลิกที่ไอคอน ⋮ มุมล่างซ้ายของ Sidebar → เลือก "ออกจากระบบ"

หรือใช้คีย์ลัด: Ctrl + Shift + L

🚫 ระบบจะออกจากระบบอัตโนมัติ: หากไม่มีการใช้งานเกิน 30 นาที

🔹 3. Roles และสิทธิ์การใช้งาน

Roleหน้าที่เมนูที่เข้าถึงได้
👑 Admin ผู้ดูแลระบบสูงสุด ทุกเมนู + จัดการผู้ใช้ + ตั้งค่าระบบ + Executive Dashboard
📄 Admin Page พนักงานขาย/แอดมินเพจ ข้อมูลลูกค้า, เคสทั้งหมด, ติดตาม, Dashboard
✅ KPI ผู้ตรวจสอบคุณภาพ ตรวจ KPI, Dashboard
🔧 Support ฝ่ายสนับสนุน Pre-Approve, เอกสาร, ส่งธนาคาร, อนุมัติ, ปิดหนี้, จำนอง, ตรวจห้อง

🔹 4. Dashboard

Dashboard เป็นหน้าแรกหลังเข้าสู่ระบบ แสดงสถิติสำคัญและข้อมูลสรุป

ข้อมูลที่แสดง

  • 📊 การ์ดสถิติ: เคสทั้งหมด, อนุมัติแล้ว, กำลังติดตาม, ยกเลิก
  • 📈 กราฟ: แนวโน้มเคสรายเดือน, KPI ภาพรวม
  • 📋 เคสล่าสุด: 5 เคสล่าสุดที่ถูกสร้าง
  • 🔔 กิจกรรม: Timeline การทำงานล่าสุด
  • 🏆 Top Performers: พนักงานที่มีผลงานดี
💡 เคล็ดลับ: คลิกที่การ์ดสถิติเพื่อกรองดูเฉพาะเคสสถานะนั้นๆ

🔹 5. การจัดการลูกค้า

การเพิ่มลูกค้าใหม่

1 ไปที่เมนู "ข้อมูลลูกค้า"

2 คลิกปุ่ม "เพิ่มลูกค้าใหม่"

3 กรอกข้อมูลตามฟอร์ม:

  • ชื่อ-นามสกุล *จำเป็น
  • เบอร์โทร *จำเป็น
  • Facebook, Line ID, เพจ
  • ช่องทาง (FB, TikTok, Line)
  • เกรด (A+, A, B)
  • โครงการที่สนใจ, ราคา, เงินทอน
  • ข้อมูลการทำงาน: บริษัท, อายุงาน, สวัสดิการ, ภาระหนี้
🔍 Auto-Fill: เมื่อกรอกเบอร์โทร ระบบจะค้นหาลูกค้าเดิมอัตโนมัติ ถ้าพบจะเติมข้อมูลให้

การส่งเคส

หลังจากบันทึกข้อมูลลูกค้า ปุ่ม "ส่งเคส" จะปรากฏขึ้น → คลิกเพื่อสร้างเคสใหม่

การค้นหาและกรอง

ใช้ช่องค้นหาและ Dropdown กรองตามช่องทาง, เกรด, โครงการ, ภาระหนี้

🔹 6. การจัดการเคส

การดูเคสทั้งหมด

ไปที่เมนู "เคสทั้งหมด" → แสดงตารางเคสทั้งหมด มีการ์ดสรุปสถานะและตัวกรอง

การเปิดรายละเอียดเคส

คลิกที่ Case ID เพื่อเปิดหน้ารายละเอียดเคส ซึ่งมี 10 Tabs สำหรับจัดการแต่ละขั้นตอน

10 Tabs ในรายละเอียดเคส

  1. 👤 ข้อมูลลูกค้า
  2. 📞 ติดตาม
  3. ✅ KPI
  4. 📋 Pre-Approve
  5. 📄 เอกสาร
  6. 🏦 ส่งธนาคาร
  7. 💰 อนุมัติ
  8. 💳 ปิดหนี้
  9. 🏠 จำนอง
  10. 🔍 ตรวจห้อง

การเปลี่ยนสถานะเคส

คลิกปุ่ม ในตารางเคส → เลือกสถานะใหม่

🔹 7. การติดตามลูกค้า

การติดตามทำผ่าน Tab "📞 ติดตาม" ในหน้ารายละเอียดเคส

การเพิ่มการติดตาม

1 ไปที่ Tab ติดตามในหน้ารายละเอียดเคส

2 คลิก "เพิ่มการติดตาม"

3 เลือกสถานะ:

  • ✅ สนใจ
  • 🌟 สนใจมาก
  • ⏳ รอดำเนินการ
  • 💬 กำลังต่อรอง
  • 🏢 นัดดูห้อง
  • 📄 ส่งเอกสารแล้ว
  • ❌ ไม่สนใจ
  • 🚫 ยกเลิก

4 บันทึกรายละเอียดและเลือกช่องทางที่ติดต่อ

5 คลิก "บันทึก"

📱 ช่องทางติดต่อ: โทรศัพท์, Line, Facebook, แชท, พบลูกค้า

🔹 8. การตรวจ KPI

สำหรับผู้ใช้ Role KPI หรือ Admin

การตรวจ KPI

1 ไปที่เมนู "ตรวจ KPI"

2 ดูรายการรอตรวจในตาราง

3 คลิก "ตรวจ" → เลือกผล ผ่าน/ไม่ผ่าน → เลือกเหตุผล

การตรวจแบบเร็ว

ใช้ช่อง "ตรวจ KPI ด่วน" ด้านขวา → กรอก Case ID → เลือกผล → บันทึก

เหตุผลที่ไม่ผ่าน

  • ไม่ตามลูกค้า
  • ลูกค้าไม่ตอบ
  • ติดบูโร
  • อายุงานไม่ถึง
  • รายได้ไม่ถึง
  • เอกสารไม่ครบ

🔹 9. Pre-Approve (อนุมัติเบื้องต้น)

สำหรับ Role Support หรือ Admin

ไปที่ Tab "📋 Pre-Approve" ในหน้ารายละเอียดเคส → เลือกสถานะ (กำลังดำเนินการ/อนุมัติ/ปฏิเสธ) → ระบุวงเงิน → บันทึก

🔹 10. เอกสาร & อัปโหลดไฟล์

สถานะเอกสาร

จัดการสถานะเอกสาร 3 ขั้นตอน: ผ่าน/ไม่ผ่าน/รอดำเนินการ

Pre-Check และสถานะปิดหนี้

การอัปโหลดไฟล์

1 เลือก ประเภทเอกสาร (บัตรประชาชน, ทะเบียนบ้าน, สลิปเงินเดือน ฯลฯ)

2 เลือกไฟล์ (รองรับ .jpg, .png, .pdf สูงสุด 5MB)

3 คลิก "อัปโหลด"

ประเภทเอกสารที่รองรับ

  • 🪪 บัตรประชาชน
  • 🏠 ทะเบียนบ้าน
  • 📄 สลิปเงินเดือน
  • 📊 Statement ย้อนหลัง 6 เดือน
  • 📝 หนังสือรับรองเงินเดือน
  • ✍️ สัญญาซื้อขาย
  • 📜 สำเนาโฉนด
  • 🏦 สำเนาบัญชีธนาคาร

🔹 11. ส่งธนาคาร

ไปที่ Tab "🏦 ส่งธนาคาร" → เลือกธนาคาร → วันที่ส่ง → หมายเหตุ → บันทึก

ระบบจะแสดงประวัติการส่งธนาคารทั้งหมด

🔹 12. ผลอนุมัติ

ไปที่ Tab "💰 อนุมัติ" → กรอกวงเงินแยกตามประเภท:

  • วงเงินห้อง
  • วงเงินประกัน
  • วงเงินเฟอร์นิเจอร์

ระบบคำนวณวงเงินรวมอัตโนมัติ

ระบุวันเซ็นสัญญาและวันโอน

📋 Copy ส่ง LINE: คลิกปุ่มสีเขียวเพื่อคัดลอกข้อความสรุปส่ง LINE

🔹 13. ปิดหนี้

ไปที่ Tab "💳 ปิดหนี้"

1 กรอกวันที่ปิดหนี้, สถานที่, เจ้าหน้าที่

2 คลิก "+" เพื่อเพิ่มรายการหนี้

3 กรอกรายละเอียดและจำนวนเงิน

4 ระบบคำนวณยอดรวมอัตโนมัติ

5 คลิก "บันทึก"

🔹 14. จำนอง

ไปที่ Tab "🏠 จำนอง" → กรอกข้อมูล: วันที่จำนอง, ธนาคาร, ชื่อบัญชี, เลขบัญชี, วงเงินอนุมัติ → บันทึก

🔹 15. ตรวจห้อง

1 ไปที่ Tab "🔍 ตรวจห้อง"

2 คลิก "เพิ่มการตรวจ"

3 เลือกครั้งที่ (1-3), วันที่ตรวจ, สถานะ (ผ่าน/ไม่ผ่าน)

4 บันทึกหมายเหตุจุดที่พบ

5 อัปโหลดรูปภาพ (สูงสุด 5 รูป) เพื่อเก็บหลักฐาน

6 คลิก "บันทึก"

📸 รูปภาพประกอบ: คลิกที่รูปเพื่อดูขนาดใหญ่ใน Lightbox

🔹 16. ผู้ดูแลระบบ (Admin)

เมนูสำหรับ Admin

  • จัดการผู้ใช้: เพิ่ม/แก้ไข/ลบ ผู้ใช้งาน
  • ข้อมูลหลัก (Master Data): จัดการ dropdown ทั้งหมด (ช่องทาง, โซน, ธนาคาร, สวัสดิการ, ประเภทเอกสาร)
  • จัดการโครงการ: เพิ่ม/แก้ไข/ลบ โครงการ
  • ตั้งค่าระบบ: ตั้งค่าทั่วไป, LINE Notify, สำรองข้อมูล
  • บันทึกระบบ: ดูประวัติการใช้งานทั้งหมด
  • Executive Dashboard: ดูภาพรวมทั้งบริษัทและ Export PDF

🔹 17. คีย์ลัด

Ctrl + Bย่อ/ขยาย Sidebar
Ctrl + Shift + Lออกจากระบบ
Escปิด Modal / Popup

🔹 18. ปัญหาที่พบบ่อย

❌ กดปุ่มแล้วไม่ทำงาน

วิธีแก้: รีเฟรชหน้า (F5) หรือออกจากระบบแล้วเข้าใหม่

❌ แสดง "Session หมดอายุ"

วิธีแก้: เข้าสู่ระบบใหม่ ระบบจะออกจากระบบอัตโนมัติหลังไม่ใช้งาน 30 นาที

❌ ภาษาไทยแสดงเป็น ????

วิธีแก้: ตรวจสอบว่าเบราว์เซอร์เข้ารหัสเป็น UTF-8

❌ ไม่สามารถอัปโหลดไฟล์ได้

วิธีแก้: ตรวจสอบขนาดไฟล์ (ไม่เกิน 5MB) และนามสกุล (.jpg, .png, .pdf)

❌ กราฟไม่แสดง

วิธีแก้: รอสักครู่หรือรีเฟรชหน้า

📞 ติดต่อผู้ดูแลระบบ: หากพบปัญหาที่ไม่สามารถแก้ไขได้เอง
